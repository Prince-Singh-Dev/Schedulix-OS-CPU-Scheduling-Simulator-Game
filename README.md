## 🧠 What is Schedulix?

**Schedulix** is a next-generation **Operating System project** that transforms CPU scheduling into an **interactive game**.

Instead of just running algorithms, you:

* Predict execution order <br>
* Visualize scheduling using Gantt Charts <br>
* Get real-time feedback <br>
* Compete for the best score <br>

---

## ✨ Features

🔥 Interactive Game Mode (5 Rounds) <br>
🎯 Real-time Scoring System <br>
📊 Gantt Chart Visualization (Terminal-Based) <br>
🧩 Mistake Analysis & Feedback <br>
🎲 Random Process Generation <br>
⚡ Supports Multiple Scheduling Algorithms <br>

---

## ⚙️ Algorithms Implemented

| Algorithm    | Description                    |
|--------------|--------------------------------|
| FCFS         | First Come First Serve         |
| SJF          | Shortest Job First (Optimal)   |
| Priority     | Based on priority value        |
| Round Robin  | Time-sharing scheduling        |

---

## 🎮 How to Play

```bash
python main.py
```

### Steps:

1. Select scheduling algorithm <br>
2. Enter execution order <br>
3. View Gantt chart <br>
4. Get score + feedback <br>
